## Image Guidelines

- **Format**: PNG preferred for screenshots
- **Resolution**: Use device resolution (1080x1920 or similar)
- **Quality**: High quality images that clearly show UI elements
- **Content**: Use sample data that represents real usage
- **Privacy**: Ensure no real personal information is visible

## How to Capture Screenshots

1. **Android Studio Emulator**: Use the camera icon in the emulator toolbar
2. **Physical Device**: Use device screenshot functionality
3. **ADB Command**: `adb shell screencap -p /sdcard/screenshot.png`

Once screenshots are added, the main README.md will automatically display them.
